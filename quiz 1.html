HTML Quiz — Answers + Why

Each option shows whether it’s correct and a short reason.

Q1Which tag is used to define a table row in HTML?

  • <td> ❌
    Defines a table data cell, not a row.
  • <tr> ✅
    Correct: <tr> wraps a table row of cells.
  • <th> ❌
    Defines a header cell within a row.
  • <table> ❌
    Defines the table container, not a row.

Q2Which tag is used to create an unordered list in HTML?

  • <ol> ❌
    Ordered list (numbered), not unordered.
  • <ul> ✅
    Correct: <ul> creates a bulleted (unordered) list.
  • <li> ❌
    List item tag; used inside <ul> or <ol>.
  • <list> ❌
    Not a valid HTML element.

Q3What is the purpose of the <mark> tag?

  • To make text bold ❌
    Bold text is <b> or <strong> (semantic).
  • To underline text ❌
    Underlining uses CSS or <u> (not recommended for links confusion).
  • To highlight text ✅
    Correct: <mark> highlights text (default yellow background).
  • To change text color ❌
    Use CSS (e.g., color) for text color.

Q4How do you define a list item in an ordered or unordered list?

  • <item> ❌
    Not an HTML tag.
  • <li> ✅
    Correct: <li> is a list item inside <ul> or <ol>.
  • <lists> ❌
    Not an HTML tag.
  • <ol> ❌
    Defines an entire ordered list, not the item.

Q5Which attribute is used to open a link in a new tab?

  • target ✅
    Correct: target="_blank" opens links in a new tab/window.
  • href ❌
    Specifies the URL for the link, not the target behavior.
  • open ❌
    Not a valid attribute on <a>.
  • newtab ❌
    Not a valid attribute.

Q6What is the purpose of the <header> tag?

  • To create a footer for a webpage ❌
    <footer> defines footer content.
  • To contain the main text of a page ❌
    <main> typically holds the primary content.
  • To make text bold ❌
    Bold is <b> or <strong>.
  • To define introductory content like logos or navigation menus ✅
    Correct: <header> is for intro/branding/nav of a page or section.

Q7How do you create a text input field in an HTML form?

  • <input type="textfield"> ❌
    No textfield type exists.
  • <input type="text"> ✅
    Correct: single-line text field.
  • <input text="field"> ❌
    Incorrect attribute; should use type.
  • <textfield> ❌
    Not a valid HTML element.

Q8Which tag is used for highlighting important text?

  • <b> ❌
    Bold only; no semantic importance.
  • <strong> ✅
    Correct: conveys importance (usually renders bold).
  • <i> ❌
    Italic/alternate voice; not “important”.
  • <span> ❌
    Generic inline container; semantic meaning depends on context/CSS.

Q9Which of the following is a semantic HTML element?

  • <div> ❌
    Non-semantic container.
  • <span> ❌
    Non-semantic inline container.
  • <article> ✅
    Correct: represents independent, self-contained content.
  • <b> ❌
    Styling only; not semantic.

Q10What does the <p> tag do in HTML?

  • Defines a paragraph ✅
    Correct: <p> wraps a paragraph block.
  • Defines a link ❌
    Links use <a>.
  • Defines a heading ❌
    Headings use <h1>–<h6>.
  • Defines a line break ❌
    Line breaks use <br>.

Q11What is the function of the <details> tag?

  • To define a short summary of content ❌
    <summary> labels the details control; not the container.
  • To create a collapsible section that the user can expand ✅
    Correct: <details> toggles hidden/visible content.
  • To display an image caption ❌
    Image captions use <figcaption>.
  • To link stylesheets ❌
    Stylesheets use <link rel="stylesheet">.

Q12Which tag is used to display an image in HTML?

  • <img> ✅
    Correct: embeds an image with src and alt.
  • <image> ❌
    Not a standard HTML element.
  • <pic> ❌
    Not a tag (there is <picture> for responsive images).
  • <src> ❌
    That’s an attribute, not an element.

Q13What is the primary purpose of semantic HTML elements?

  • To improve SEO and accessibility by giving meaning to content. ✅
    Correct: semantics help search engines and assistive tech understand structure.
  • To make HTML code shorter ❌
    They add meaning, not necessarily brevity.
  • To increase website speed ❌
    Performance depends on many other factors.
  • To replace JavaScript functionality ❌
    Semantics ≠ scripting.

Q14Which of the following is not a semantic tag?

  • <article> ❌
    Semantic: standalone, self-contained content.
  • <header> ❌
    Semantic: introductory content for page/section.
  • <section> ❌
    Semantic: thematic grouping of content.
  • <div> ✅
    Correct: generic, non-semantic container.

Q15Which tag is best for grouping related content in a webpage?

  • <section> ✅
    Correct: groups related content under a theme.
  • <nav> ❌
    Specifically for navigation links.
  • <header> ❌
    Introductory content, not general grouping.
  • <footer> ❌
    Footer area; not for grouping arbitrary content.

Q16What does the <h1> tag do in HTML?

  • Defines a paragraph ❌
    Paragraphs use <p>.
  • Defines a link ❌
    Links use <a>.
  • Defines the largest heading ✅
    Correct: <h1> is the top-level heading.
  • Defines an image ❌
    Images use <img>.

Q17Which HTML element is used to define navigation links in a webpage?

  • <nav> ✅
    Correct: wraps blocks of site navigation links.
  • <header> ❌
    Intro section; may contain <nav> but isn’t navigation itself.
  • <section> ❌
    General thematic grouping; not specifically navigation.
  • <aside> ❌
    Tangential content (e.g., sidebars), not navigation.

Q18Which attribute is necessary to specify the URL of an image in HTML?

  • alt ❌
    Alternative text; not the image URL.
  • href ❌
    URL for links; images use src.
  • src ✅
    Correct: specifies the image source path/URL.
  • link ❌
    Not an attribute for <img>.

Q19Which tag is used to create an ordered list in HTML?

  • <ol> ✅
    Correct: ordered (numbered) list.
  • <ul> ❌
    Unordered (bulleted) list.
  • <li> ❌
    List item; used inside <ol>/<ul>, not the list itself.
  • <list> ❌
    Not a valid HTML tag.

Q20Which tag is used to create a form in HTML?

  • <form> ✅
    Correct: container for form controls and submission.
  • <input> ❌
    Form control used inside a form.
  • <button> ❌
    Clickable button; often submits a form but isn’t the form itself.
  • <textarea> ❌
    Multi-line text input; belongs inside a form.
