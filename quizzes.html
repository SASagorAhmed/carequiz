1) Which tag creates an unordered list?

Correct: <ul> makes a bulleted list. Others are wrong because: <ol> = ordered list, <li> = list item, <list> = invalid.

2) Which attribute opens a link in a new tab?

Correct: target="_blank". href sets the URL; the others are not valid attributes.

3) What does <h1> represent?

Correct: Largest/most important heading. Paragraph = <p>, Link = <a>, Image = <img>.

4) Which element defines navigation links?

Correct: <nav> groups navigation links. Others are layout/semantic but not navigation containers.
