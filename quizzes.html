HTML Quiz

Select the answers and click “Check Score”.

1) Which tag is used to define a table row in HTML?

Correct: <tr> defines a table row.

2) Which tag is used to create an unordered list?

Correct: <ul> defines an unordered list.

3) What is the purpose of the <mark> tag?

Correct: <mark> highlights text (yellow by default).

4) How do you define a list item?

Correct: <li> defines a list item.

5) Which attribute opens a link in a new tab?

Correct: target="_blank" opens a link in a new tab.

6) What is the purpose of the <header> tag?

Correct: <header> defines introductory content.

7) How do you create a text input field?

Correct: <input type="text">.

8) Which tag highlights important text?

Correct: <strong> marks important text.

9) Which is a semantic element?

Correct: <article> is semantic.

10) What does <p> tag do?

Correct: <p> defines a paragraph.

11) Function of <details> tag?

Correct: <details> makes collapsible content.

12) Which tag displays an image?

Correct: <img>.

13) Purpose of semantic elements?

Correct: Semantics improve SEO & accessibility.

14) Which is NOT semantic?

Correct: <div> is not semantic.

15) Best tag for grouping related content?

Correct: <section>.

16) What does <h1> tag do?

Correct: <h1> is the largest heading.

17) Which element defines navigation links?

Correct: <nav>.

18) Which attribute specifies image URL?

Correct: src specifies image path.

19) Which tag creates an ordered list?

Correct: <ol> defines an ordered list.

20) Which tag is used to create a form?

Correct: <form> defines a form.
